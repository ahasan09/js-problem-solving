# Problem Practice

Vanilla JavaScript practice problems including a number-to-words converter, built with plain HTML/CSS/JS.

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
```
problem-practice/
├── index.html
├── style.css
└── index.js
```

## Development
No installation required. Open `index.html` directly in a browser:
```bash
open problem-practice/index.html
```

## Key Notes
- No build step or dependencies — fully self-contained static files.
