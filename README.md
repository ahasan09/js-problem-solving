# JS Problem Solving

A browser-based JavaScript app that converts numbers into their English word representation.

## Demo

Open `index.html` in any modern browser to see a table of numbers rendered as English words.

## Prerequisites

- Any modern web browser (Chrome, Firefox, Edge, Safari)

## Running

No installation needed — just open the file:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Or serve locally:

```bash
npx serve .
```

Then open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
js-problem-solving/
├── index.html   # Entry point — renders the number table
├── index.js     # Number-to-words conversion logic
└── style.css    # Styles
```
